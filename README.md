# 🏦 A股优选分红股票监测面板

> 一句话介绍：**优选高分红A股股票的实时买入信号监测面板，绿/黄/红三色提示，按多类估值框架判断，大白话告诉你现在该买还是等。**

全套系统完全免费，部署一次，之后每天自动运行，打开网页就能看。


## 一、这个系统能干什么？

| 功能 | 说明 |
|------|------|
| 📡 实时监测 | 每天 10:00 和 14:00（北京时间）自动更新两次 |
| 🟢🟡🔴 买入信号 | 绿色=可买，黄色=观望，红色=不建议，按四类估值框架判断 |

| 📱 手机/电脑通用 | 自适应屏幕，随时随地查看 |
| 📢 飞书通知 | 支持迟到警告、更新成功/失败通知（需配置 Webhook） |

## 四、🚀 0基础5步部署到 GitHub Pages

> 不需要懂代码，不需要买服务器，全程免费。跟着做，10分钟搞定。

---

### 第1步：注册/登录 GitHub

- 打开 [github.com](https://github.com)，注册账号（已有则直接登录）
- 验证邮箱（必须，否则无法创建仓库）

---

### 第2步：新建仓库

1. 点击页面右上角 `+` → `New repository`
2. 仓库名填：`stock-monitor`（可以自己取别的名字）
3. 选择 **Public**（必须选 Public，否则 GitHub Pages 不可用）
4. 勾选 **Add a README file**（方便后续编辑）
5. 点击 **Create repository**

---

### 第3步：上传所有文件

进入刚创建的仓库，上传以下文件（点 `Add file` → `Upload files`）：

| 要上传的文件 | 说明 |
|-------------|------|
| `stock_monitor.py` | Python数据抓取脚本（核心） |
| `index.html` | 可视化网页面板 |
| `requirements.txt` | Python依赖清单 |
| `forward_dividend.json` | 前瞻分红数据 |

> **💡 小技巧**：`update_data.yml` 需要放在 `.github/workflows/` 目录下。
> 最简单的方式：把本项目的所有文件**拖拽到 GitHub 仓库页面**，GitHub 会自动创建对应的文件夹结构。

---

### 第4步：开启 GitHub Pages

1. 进入仓库页面，点击顶部 **Settings**

> ⚠️ 注意：必须选择 "GitHub Actions" 而不是 "Deploy from a branch"，
> 因为本系统使用 GitHub Actions 自动生成数据文件。

### 第5步：等5分钟，访问你的监测面板！

1. 进入仓库 **Actions** 标签页，可以看到工作流正在运行（绿色圆点 = 运行中）

## 五、飞书通知配置（可选）

系统支持通过飞书自定义机器人发送通知，需配置 GitHub Secrets：

### 第1步：创建飞书自定义机器人
1. 打开飞书群 → 右上角 `...` → **群机器人** → **添加机器人**
2. 选择 **自定义机器人** → 起个名字（如"股票监测"）→ 添加
3. 复制 **Webhook 地址**（格式：`https://open.feishu.cn/open-apis/bot/v2/hook/xxxx`）

### 第2步：存储到 GitHub Secrets
1. 打开：https://github.com/你的用户名/仓库名/settings/secrets/actions
2. 点击 **New repository secret**
3. Name：`FEISHU_WEBHOOK`
4. Secret：粘贴飞书 Webhook 地址
5. 点击 **Add secret**

### 通知类型
| 类型 | 触发条件 |
|------|---------|
| 迟到警告 | 比预定时间晚30分钟以上（GitHub Actions 队列延迟） |
| 更新成功 | 数据更新完成并成功推送 |
| 更新失败 | 脚本运行出错（需检查日志） |

## 七、买入信号是怎么算出来的？

系统按 **四类估值框架** 分类计算，每类股票使用独立评分标准：

### 1. 银行股（硬门槛 + 加分项）
- **硬门槛**：必须满足 PE<5 或 PB<0.65 才进入评分（避免招商银行式误判）
- **加分项**：股息率>5%、接近目标股息率、分红季（6-7月）、大盘恐慌
- **总分≥50** → 绿色（可买入）

### 2. 强周期能源（石油看PB，煤炭看股息率）
- 中国石油/中国石化：PB≤1.0 极佳买点，PB>1.3 高估区间
- 中国神华：股息率≥6.5% 极佳买点，股息率<4.5% 高估区间
- 陕西煤业：股息率≥6.0% 极佳买点，股息率<4.0% 高估区间
- 辅助：国际油价60~80美元时盈利最稳；煤价600~800元/吨时盈利最稳

### 3. 公用事业（类债券资产，看股息率+PE）
- 中国移动：股息率≥5.5% 或 PE≤12 极佳买点
- 长江电力：PE≤16 且股息率≥4.5% 极佳买点
- 宁沪高速：股息率≥6.5% 或 PE≤10 极佳买点
- 申能股份：股息率≥4.5% 或 PE≤10 极佳买点

### 4. 消费白马 / 保险 / 医药红利股 / 高端制造成长
- 贵州茅台：前瞻PE≤18 且股息率≥4% 极佳买点
- 美的集团：PE≤11 且股息率≥5.5% 极佳买点
- 双汇发展：PE≤15 且股息率≥5% 极佳买点
- 伊利股份：PE≤18 且股息率≥4% 极佳买点
- 格力电器：PE≤8 且股息率≥5% 极佳买点
- 华润江中：股息率≥4.5% 极佳买点
- 济川药业：股息率≥5.0% 极佳买点
- 云南白药：股息率≥4.0% 极佳买点
- 宁德时代：前瞻PE≤20 且 PEG≤0.8 极佳买点
- 中国建筑：前瞻PE≤5 且 PEG≤0.8 极佳买点
- 宇通客车：前瞻PE≤12 且 PEG≤0.8 极佳买点

> **大白话解释**：
> - 🟢 绿色信号 = 达到"极佳买点"条件，胜率高，可以分批买入
> - 🟡 黄色信号 = 达到"合理买点"，可小仓位关注，不急于全仓
> - 🔴 红色信号 = 当前不是好买点，或估值未达安全区，耐心等待更好机会

## 九、常见问题 FAQ

**Q：为什么 data.json 没有自动生成？**
A：进入仓库 Actions 页面查看工作流运行日志，常见原因是 yfinance 访问受限，可尝试在本地运行 `python stock_monitor.py` 生成后手动推送。

**Q：可以自己添加其他股票吗？**
A：可以！编辑 `stock_monitor.py` 顶部的 `STOCK_MAP`，按同样格式添加，并在 `forward_dividend.json` 中补充每股分红数据，然后提交即可。

**Q：GitHub Actions 运行时间不准？**
A：GitHub Actions 的 schedule 触发器是尽力而为（best-effort），免费 tier 在高负载时会有延迟。已配置迟到检测，超过30分钟会发飞书警告。

**Q：银行股为什么显示"不建议"？**
A：银行股设有硬门槛：必须满足 PE<5 或 PB<0.65 才进入评分，避免股息率高但估值不安全的误判（如招商银行）。

**Q：GitHub Pages 访问不了？**
A：国内访问 `github.io` 可能需要科学上网，或者可以部署到国内的代码托管平台（如 Gitee Pages）。

**Q：这个系统能保证赚钱吗？**
A：不能。本系统仅提供数据监测和操作参考，**不构成任何投资建议**。股市有风险，投资需谨慎。
