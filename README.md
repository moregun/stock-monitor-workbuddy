## 四、🚀 0基础5步部署到 GitHub Pages

> 不需要懂代码，不需要买服务器，全程免费。跟着做，10分钟搞定。

---

### 第1步：注册/登录 GitHub

- 打开 [github.com](https://github.com)，注册账号（已有则直接登录）
- 验证邮箱（必须，否则无法创建仓库）

---

### 第2步：新建仓库

1. 点击页面右上角 `+` → `New repository`
2. 仓库名填：`stock-monitor`（可以自己取别的名字）
3. 选择 **Public**（必须选 Public，否则 GitHub Pages 不可用）
4. 勾选 **Add a README file**（方便后续编辑）
5. 点击 **Create repository**

---

### 第3步：上传4个文件

进入刚创建的仓库，依次上传以下文件（点 `Add file` → `Upload files`）：

| 要上传的文件 | 本地路径 |
|-------------|---------|
| `stock_monitor.py` | 本项目根目录的 `stock_monitor.py` |
| `index.html` | 本项目根目录的 `index.html` |
| `requirements.txt` | 本项目根目录的 `requirements.txt` |
| `.github/workflows/update_data.yml` | 注意：这个文件路径包含文件夹，上传时先新建文件夹 |

> **💡 小技巧**：`update_data.yml` 需要放在 `.github/workflows/` 目录下。
> 上传时在仓库首页新建文件夹 `.github`，再在里面新建 `workflows`，最后上传文件。

或者直接把本项目的所有文件**拖拽到 GitHub 仓库页面**，GitHub 会自动创建对应的文件夹结构。

---

### 第4步：开启 GitHub Pages

1. 进入仓库页面，点击顶部 **Settings**
2. 左侧菜单找到 **Pages**（在 "Code and automation" 分类下）
3. **Build and deployment** → **Source** 选择 **GitHub Actions**
4. 页面会自动保存，无需其他操作

> ⚠️ 注意：必须选择 "GitHub Actions" 而不是 "Deploy from a branch"，
> 因为本系统使用 GitHub Actions 自动生成数据文件。

---

### 第5步：等5分钟，访问你的监测面板！

1. 进入仓库 **Actions** 标签页，可以看到工作流正在运行（绿色圆点 = 运行中）
2. 第一次运行需要约 **3-5分钟**（要安装 Python 依赖）
3. 运行完成后，访问：
   ```
   https://你的用户名.github.io/stock-monitor/
   ```
4. 看到面板页面 = 部署成功 🎉

## 六、买入信号是怎么算出来的？

系统同时监测 **4大买入窗口**，任意满足即提示买入：

| 窗口 | 触发条件 | 优先级 |
|------|---------|--------|
| 🔥 大盘恐慌大跌 | 沪深300单日跌幅＞2.5% | 最高（黄金买点） |
| 📅 年底建仓期 | 当前月份是10月/11月/12月 | 高 |
| 🎁 分红前布局 | 银行/能源通信：4-5月；宁德时代：3-4月 | 中 |
| 💎 估值安全区间 | 银行PE＜5且PB＜0.65；能源通信股息率＞6%；保险PEV＜0.7；宁德时代PE＜15 | 基础 |

> **大白话解释**：
> - 🟢 绿色信号 = 多个窗口同时触发，胜率高，可以分批买入
> - 🟡 黄色信号 = 有初步信号，可小仓位关注，不急于全仓
> - 🔴 红色信号 = 当前不是好买点，耐心等待更好机会

## 八、常见问题 FAQ

**Q：为什么 data.json 没有自动生成？**
A：进入仓库 Actions 页面查看工作流运行日志，常见原因是 akshare 版本问题，可尝试在 `requirements.txt` 中指定版本。

**Q：可以自己添加其他股票吗？**
A：可以！编辑 `stock_monitor.py` 顶部的 `STOCK_LIST`，按同样格式添加，然后提交即可。

**Q：GitHub Pages 访问不了？**
A：国内访问 `github.io` 可能需要科学上网，或者可以部署到国内的代码托管平台（如 Gitee Pages）。

**Q：这个系统能保证赚钱吗？**
A：不能。本系统仅提供数据监测和操作参考，**不构成任何投资建议**。股市有风险，投资需谨慎。
